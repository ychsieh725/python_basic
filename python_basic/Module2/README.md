## 📊 基本型態 Primitive Types

| 類型 | 說明 | 創建方式 | 範例 |
|:---:|:---|:---|:---|
| 🔢 `int` | 整數 | `num = 10` | `1`, `-5`, `100` |
| 🔣 `float` | 浮點數 | `num = 10.5` | `3.14`, `-0.5` |
| ✅ `bool` | 布林值 | `is_valid = True` | `True`, `False` |
| 📝 `str` | 字串 | `name = "Jarvis"` | `"Hello"`, `'World'` |

---

## 🗃️ 容器型態 Container Types

| 類型 | 說明 | 創建方式 | 特性 |
|:---:|:---|:---|:---|
| 📋 `list` | 列表 | `nums = [1, 2, 3]` | 有序、可變、可重複 |
| 📦 `tuple` | 元組 | `nums = (1, 2, 3)` | 有序、不可變、可重複 |
| 🎯 `set` | 集合 | `nums = {1, 2, 3}` | 無序、可變、不可重複 |
| 📖 `dict` | 字典 | `data = {"a": 1}` | 鍵值對、可變 |

## ➕ 運算子 Operators

### 🧮 算術運算子
| 運算子 | 說明 | 範例 | 結果 |
|:---:|:---|:---|:---:|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 減法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法 | `5 / 3` | `1.666...` |
| `//` | 整數除法 | `5 // 3` | `1` |
| `%` | 取餘數 | `5 % 3` | `2` |
| `**` | 次方 | `5 ** 3` | `125` |

### 🔄 賦值運算子
| 運算子 | 範例 | 等同於 |
|:---:|:---|:---|
| `=` | `x = 5` | 賦值 |
| `+=` | `x += 3` | `x = x + 3` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 3` | `x = x * 3` |

### ⚖️ 比較運算子
| 運算子 | 說明 | 範例 |
|:---:|:---|:---|
| `==` | 等於 | `5 == 5` → `True` |
| `!=` | 不等於 | `5 != 3` → `True` |
| `>` | 大於 | `5 > 3` → `True` |
| `<` | 小於 | `5 < 3` → `False` |
| `>=` | 大於等於 | `5 >= 5` → `True` |
| `<=` | 小於等於 | `5 <= 3` → `False` |

### 🔗 邏輯運算子
| 運算子 | 說明 | 範例 |
|:---:|:---|:---|
| `and` | 且 | `True and False` → `False` |
| `or` | 或 | `True or False` → `True` |
| `not` | 非 | `not True` → `False` |

### 🔍 成員運算子
| 運算子 | 說明 | 範例 |
|:---:|:---|:---|
| `in` | 存在於 | `3 in [1,2,3]` → `True` |
| `not in` | 不存在於 | `4 not in [1,2,3]` → `True` |

## 💡 小技巧：判斷奇偶數

| 條件 | 表達式 | 說明 |
|:---:|:---|:---|
| 偶數 | `n % 2 == 0` | 除以 2 餘數為 0 |
| 奇數 | `n % 2 != 0` | 除以 2 餘數不為 0 |
