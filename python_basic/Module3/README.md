## 🔀 條件判斷 Conditional Statements

> [!IMPORTANT]
> 要注意條件的判斷順序，避免邏輯錯誤

### 📋 語法結構

```python
# 單一條件
if 條件:
    執行程式碼

# 二選一
if 條件:
    執行程式碼A
else:
    執行程式碼B

# 多重條件
if 條件1:
    執行程式碼A
elif 條件2:
    執行程式碼B
elif 條件3:
    執行程式碼C
else:
    以上條件都未符合，要執行的程式碼
```

## 🔄 迴圈 Loops

### 📋 for 搭配 `range()`

| 用法 | 說明 | 範例 |
|:---|:---|:---|
| `range(m)` | 0 到 m-1 | `range(5)` → 0,1,2,3,4 |
| `range(n, m)` | n 到 m-1 | `range(2, 5)` → 2,3,4 |
| `range(n, m, step)` | n 到 m-1，間距 step | `range(0, 10, 2)` → 0,2,4,6,8 |

```python
for i in range(5):
    print(i)  # 輸出 0, 1, 2, 3, 4
```

### 📋 while 迴圈

```python
count = 0
while count < 5:
    print(count)
    count += 1
```

## ⏹️ 迴圈控制

| 關鍵字 | 功能 | 使用時機 |
|:---:|:---|:---|
| `break` | 🛑 強制結束迴圈 | 達到特定條件時立即跳出迴圈(不論剩幾次) |
| `continue` | ⏭️ 跳過本次迭代 | 跳過 continue 以下的程式碼，繼續執行下一輪迴圈 |

## 🎮 練習專案：終極密碼

### 📋 遊戲規則
設計一個猜數字遊戲，讓使用者猜測隨機產生的數字

### 🎯 需求規格

| 編號 | 需求描述 |
|:---:|:---|
| 1️⃣ | 顯示需要輸入的數字範圍 |
| 2️⃣ | 猜對顯示「🎉 恭喜中獎」並結束遊戲 |
| 3️⃣ | 超出範圍顯示「⚠️ 超出範圍請重新輸入」 |
| 4️⃣ | 數字太大顯示「📉 請輸入更小的數字」 |
| 5️⃣ | 數字太小顯示「📈 請輸入更大的數字」 |

### 💻 預期輸出範例
```
請輸入 1 到 100 之間的數字: 50
📉 請輸入更小的數字
請輸入 1 到 50 之間的數字: 25
📈 請輸入更大的數字
請輸入 25 到 50 之間的數字: 37
🎉 恭喜中獎！答案就是 37
```

### 💡 提示
- 使用 `random.randint()` 產生隨機數
- 使用 `while` 迴圈持續進行遊戲
- 動態更新數字範圍
