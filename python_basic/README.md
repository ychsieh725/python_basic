# 🐍 Python 基礎教學

> 從零開始學習 Python 程式設計的完整教學課程

---

## 📚 課程總覽

本課程涵蓋 Python 程式設計的核心概念，從基本資料型態到進階的物件導向程式設計，循序漸進地帶你掌握 Python 開發技能。

---

## 🗂️ 課程模組

| 模組 | 主題 | 內容概述 | 難度 |
|:---:|:---|:---|:---:|
| 📦 [Module2](./Module2/) | **資料型態** | int, float, bool, string, list, tuple, set, dict | 🟢 入門 |
| 🔀 [Module3](./Module3/) | **流程控制** | if 條件判斷, for/while 迴圈, break/continue | 🟢 入門 |
| 🗃️ [Module4](./Module4/) | **資料結構** | 字串處理, 串列操作, 字典應用, 切片技巧 | 🟡 基礎 |
| ⚙️ [Module5](./Module5/) | **函式** | 函式定義, 參數傳遞, 變數範圍, Lambda | 🟡 基礎 |
| 🏗️ [Module6](./Module6/) | **物件導向** | 類別設計, 繼承, 封裝 | 🟠 進階 |

| 📁 [Module8](./Module8/) | **檔案處理** | 檔案讀寫, JSON 處理, OS 模組 | 🟡 基礎 |

## 📋 難度說明

## 🚀 開始學習

1. 確保已安裝 Python 3.x
2. 建議使用 VS Code 或 Jupyter Notebook 進行練習
3. 依照模組順序學習，完成每個模組的練習題

---

## 📝 檔案類型說明

| 副檔名 | 說明 | 使用方式 |
|:---:|:---|:---|
| `.ipynb` | Jupyter Notebook | 互動式學習與練習 |
| `.py` | Python 腳本 | 獨立執行的練習程式 |
